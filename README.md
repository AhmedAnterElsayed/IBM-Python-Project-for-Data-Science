# IBM Python Project for Data Science

## Project Overview

This project was completed as part of the IBM **Python Project for Data Science** course on Coursera.

The objective is to analyze historical stock prices and revenue data for **Tesla (TSLA)** and **GameStop (GME)** using Python, web scraping, and data visualization.

## Objectives

- Extract historical stock price data using the `yfinance` library.
- Extract company revenue data from HTML tables using web scraping.
- Clean and prepare the datasets using Pandas.
- Visualize stock prices and revenue using Python.

## Technologies Used

- Python
- Pandas
- yfinance
- requests
- BeautifulSoup (bs4)
- html.parser
- matplotlib

## Project Tasks

### Question 1
Extract Tesla historical stock data using `yfinance`.

### Question 2
Extract Tesla revenue data using web scraping.

### Question 3
Extract GameStop historical stock data using `yfinance`.

### Question 4
Extract GameStop revenue data using web scraping.

### Question 5
Create a visualization of Tesla stock price and revenue.

### Question 6
Create a visualization of GameStop stock price and revenue.
